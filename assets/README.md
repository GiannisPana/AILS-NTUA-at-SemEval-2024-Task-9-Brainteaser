# Assets

Figures in this directory are hand-coded SVG charts derived from the AILS-NTUA
SemEval-2024 Task 9 paper tables.

| File | Description |
|---|---|
| `our-best-vs-baselines.svg` | README headline chart comparing human, AILS-NTUA best systems, ChatGPT, and vanilla RoBERTa-large on SP/WP. |
| `small-beats-big.svg` | Matched LoRA-config comparison between Mistral-7B and Mixtral-8x7B. |
| `lora-exploration.svg` | Two-panel summary of QLoRA configurations reported in Tables 4 and 5. |
